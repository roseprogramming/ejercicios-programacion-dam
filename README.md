# Ejercicios de Programación DAM

Este proyecto contiene una serie de ejercicios y prácticas de programación en Java, organizados en diferentes categorías y conceptos. Está diseñado para ayudar a los estudiantes a aprender y practicar habilidades de programación fundamentales y avanzadas.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- **pom.xml**: Archivo de configuración de Maven que define las dependencias y configuraciones necesarias para compilar y ejecutar el proyecto.
- **README.md**: Documentación del proyecto que incluye una descripción, instrucciones de instalación y uso.
- **.gitignore**: Especifica qué archivos y directorios deben ser ignorados por Git.
- **src/**: Contiene el código fuente de la aplicación.
  - **main/java/com/dam/programacion/**: Código fuente principal.
    - **fundamentos/**: Conceptos básicos de programación.
      - **operadores/**: Clases relacionadas con operadores en Java.
      - **estructuras_control/**: Clases que implementan estructuras de control.
      - **entrada_salida/**: Clases que manejan la entrada y salida de datos.
    - **arrays/**: Clases para trabajar con arreglos.
      - **unidimensionales/**: Clases que manejan arreglos unidimensionales.
      - **bidimensionales/**: Clases que manejan arreglos bidimensionales.
    - **poo/**: Conceptos de programación orientada a objetos.
      - **clases_basicas/**: Clases que representan conceptos básicos de POO.
      - **herencia/**: Clases que demuestran el concepto de herencia.
      - **polimorfismo/**: Clases que demuestran el polimorfismo.
    - **proyectos/**: Diferentes proyectos.
      - **urgencias/**: Clases relacionadas con un proyecto de urgencias.
      - **animales/**: Clases que representan diferentes animales.
      - **carreras/**: Clases relacionadas con un proyecto de carreras.
      - **superheroes/**: Clases que representan diferentes superhéroes.
  - **test/java/com/dam/programacion/**: Pruebas unitarias del proyecto.
- **docs/**: Documentación adicional.
  - **apuntes/**: Apuntes y notas relacionadas con el proyecto.
  - **practicas/**: Prácticas y ejercicios relacionados con el proyecto.

## Instrucciones de Instalación

1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Ejecuta `mvn install` para compilar el proyecto y descargar las dependencias necesarias.

## Uso

Puedes ejecutar las clases desde el directorio `src/main/java/com/dam/programacion/` según el concepto que desees practicar. Asegúrate de tener Java y Maven instalados en tu sistema.

¡Feliz programación!
